# Liberty Gas & Deli Website

Professional business website for **Liberty Gas & Deli** in Elkton, Virginia.

## Business Info

**Name:** Liberty Gas & Deli  
**Address:** 607 7th St, Elkton, VA 22827  
**Website Purpose:** Showcase fuel, deli food, hours, location, menu items, and online ordering.

## Features

- Modern responsive homepage
- Deli menu with real items and prices
- Breakfast, lunch, fried chicken, and pizza sections
- DoorDash ordering link
- Google Maps directions
- Location and hours section
- Holtzman-supplied fuel mention
- Mobile-friendly layout
- Professional gas station + deli branding

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel deployment
